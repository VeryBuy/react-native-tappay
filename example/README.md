# react-native-tappay example

## Usage

Install dependencies first:

```
$ npm install // or yarn
```

Run web:

```
$ npm run web // or yarn web
```

Run iOS:


```
$ cd ios
$ pod install
$ cd ..
$ npm run ios // or yarn ios
```

Run Android:


```
$ npm run android // or yarn android
```
